
**Project Description**

This is a personal fraud detection project developed from a very large and highly imbalanced dataset.

**Model Choice**

After testing several models, XGBoost proved to be the most effective thanks to its performance, flexibility, and interpretability.

**Methodical Approach**

The work follows a structured workflow:
- data analysis
- feature engineering
- dataset balancing
- model training
- performance evaluation

**Value of Domain Knowledge**

Professionals working in fraud detection have an advantage, as they can leverage more specific data and deep knowledge of behavioral patterns and typical fraud schemes.

**Code Implementation**

For clarity and simplicity, the code was written inline without using pipelines, making the logic immediately readable.


> To run the code efficiently, it is recommended to use a T4 GPU, which accelerates model training and large-scale data processing.
